# 更換核心

您想要更換伺服器核心嗎？不需要重創伺服器！

<Callout type="info">本教學僅適用於 Minecraft Java 版。</Callout>
<Callout type="warning">做任何動作前，請確保您有保存/備份好檔案。</Callout>

<Steps>
### 下載核心
首先，您需要下載您要的伺服器核心。
<Tabs items={["Vanilla","Paper","Purpur","Bungeecord","其他"]}>
  <Tabs.Tab>**Vanilla** 是 Minecraft 原版核心，您可以在 [這裡](https://mcversions.net/) 下載它的核心檔案。</Tabs.Tab>
  <Tabs.Tab>**Paper** 是 Minecraft 插件核心，您可以在 [這裡](https://papermc.io/) 下載它的核心檔案。</Tabs.Tab>
  <Tabs.Tab>**Purpur** 是 Minecraft 插件核心，您可以在 [這裡](https://purpurmc.org/) 下載它的核心檔案。</Tabs.Tab>
  <Tabs.Tab>
  **Bungeecord** 是 Minecraft Proxy 核心，您可以在 [這裡](https://ci.md-5.net/job/BungeeCord/) 下載它的核心檔案。
  <Callout type="info">[點此查看我們為 Bungeecord 出的教學！](/panel/practice/bungeecord)</Callout>
  </Tabs.Tab>
  <Tabs.Tab>沒有列出您想要的嗎？想辦法得到它的核心檔案就好了。</Tabs.Tab>
</Tabs>

### 刪除舊核心

到 Files 把 `server.jar` 刪掉就好了，如果您沒有更改名稱的話...

### 上傳核心

把您下載下來的核心丟到 Files 頁面中，並且更改名稱為 `server.jar`。

### 啟動設定

請根據您的伺服器版本選擇 Java 版本，請參考[這篇文章](https://docs.mcserversoft.com/advanced/java-version)。

</Steps>
