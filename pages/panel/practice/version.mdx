# 更換版本

您想要更換伺服器版本嗎？不需要重創伺服器！

<Callout type="info">本教學僅適用於 Minecraft Java 版。</Callout>
<Callout type="warning">做任何動作前，請確保您有保存/備份好檔案。</Callout>

<Steps>
### 到 Startup 頁面

他會有很多個空格讓你填，找到這個。
![啟動變數](/panel/change-version/variables.png)

---

### 編輯 MINECRAFT VERSION

就直接輸入版本號就好了。

### 重裝伺服器

到 Settings，點選 Reinstall Server。
![重裝伺服器](/panel/change-version/reinstall.png)

### 等待

等它跑玩就好了，可能會很久，也有可能很快！

</Steps>
