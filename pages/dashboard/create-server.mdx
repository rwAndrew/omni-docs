# 創建伺服器

## 在您登入以後

<Steps>
### 請至 [資源頁面](https://dash.shdcloud.xyz/dashboard)
### 點擊 [創建伺服器](https://dash.shdcloud.xyz/dashboard/create)
### 設定伺服器
- 伺服器名稱
- 節點
	- 台灣
	- 美國
- 伺服器類型
- CPU（單位為%）
- 記憶體（單位為 MiB）
- 儲存空間（單位為 MiB）
<Callout type="warning">請確實填寫</Callout>

### 點創建按鈕

### 完成了，好欸

</Steps>

## 創建完以後

<Steps>
  ### 回到[主頁](https://dash.shdcloud.xyz/dashboard) ###
  找到您創建好的伺服器，並按下管理 ### 按下控制，並登入面板
  (請參考這篇[文章](https://docs.shdcloud.xyz/panel/login))
</Steps>
